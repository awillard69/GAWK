GAWK by awillard69

Modified GAWK to support embedded delimiters, clean compile on CYGWIN.

This is my variation of the venerable GAWK program. I need the program to support additional features not available in the typical release. Features such as field embedded delimiters is the main reason for this variation.

Install

Download sources. Run ./configure, then "make && make check".
